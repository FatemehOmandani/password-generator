# Random Password Generator

## Description
This is a simple project that generates random and secure passwords. You can select the length of the password and easily copy it to your clipboard.

## How to Use
1. Choose the length of the password from the dropdown.
2. Click "Generate passwords" to create two random passwords.
3. Click on a password to copy it to your clipboard.

## Project Structure
- `index.html`: The main webpage structure.
- `index.css`: Styles for the page.
- `index.js`: Logic for generating passwords and copying them.

## Requirements
- Any modern web browser (Chrome, Firefox, etc.)
